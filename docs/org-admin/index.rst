************************************************************
Organization Administrator Guide
************************************************************

Administrators of an organization can:

* Create new users within the organization
* Assign permissions to users
* View all the organiation's catalog records
* Assign curators to catalog records
* Approve catalog record publication requests
* Edit the organization's name and other settings

This guide contains instructions for organization administrators.
